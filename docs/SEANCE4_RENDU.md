# Rendu Séance 4

## Intitulé

Composition Three.js du projet `banapoulpa`.

## Contenu livré

- scène Three.js finale dans `src/` avec `index.html` à la racine
- export de scène `public/assets/banapoulpa/scene.json`
- modèle animé `public/assets/banapoulpa/bananapoulpa.glb`
- configuration Vite pour lancer et builder la scène
- vidéo de rendu `bananapoulpa.mp4`

## Fonctionnalités intégrées

- chargement de la composition depuis un export JSON Three.js
- remplacement du poulpe statique par le GLB animé
- lecture en boucle des animations
- caméra finale configurée pour la présentation
- ambiance de laboratoire : fumée, bloom, lumières chimiques, néons
- audio d'ambiance activable dans la scène

## Lancement

```bash
npm install
npm run dev
```
